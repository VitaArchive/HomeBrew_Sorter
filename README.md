## File extensions supported
- ISO
- CSO
- ZSO
- JSO
- DAX
- PBP (EBOOT)

## Current detected paths
- `ms0:/PSP/GAME`
- `ms0:/PSP/GAME/PSX`
- `ms0:/PSP/GAME/Utility`
- `ms0:/PSP/GAME150`
- `ms0:/ISO`
- `ms0:/ISO/PSP`
- `ms0:/ISO/VIDEO`
